Google スプレッドシート自動連携

以下のコードをApps Scriptに貼り付けると、保存時にスプレッドシートへ自動追記されます。

function doPost(e) { var ss = SpreadsheetApp .getActiveSpreadsheet(); var sh = ss.getSheetByName("実績") || ss.insertSheet("実績"); var d = JSON.parse(e.postData.contents); sh.appendRow([ d.date, d.weight, d.steps, d.burned, d.intake, d.breakfast, d.lunch, d.dinner, d.training, new Date() ]); return ContentService .createTextOutput("OK"); }
  • スプレッドシートを開く → 拡張機能 → Apps Script
  • 上記コードを貼り付けて保存
  • デプロイ → 新しいデプロイ → ウェブアプリ
  • アクセス権を「全員」にしてデプロイ
  • 発行されたURLを下に入力
ホーム画面への追加方法
  • このHTMLファイルをサーバーやNetlifyにアップロード
  • iPhoneはSafariで開く → 共有 → 「ホーム画面に追加」
  • Androidはクロームで開く → メニュー → 「ホーム画面に追加」

Netlify（netlify.com）は無料でHTMLをアップロードしてURLを発行できます。ドラッグ＆ドロップでOKです。
